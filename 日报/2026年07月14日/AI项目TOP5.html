01
goose — 可执行、可扩展的本地 AI Agent
⭐ ~51K
是什么：Block（前 Square）出品的开源本地 AI Agent 框架——Rust 单二进制，定位「不只给代码补全，而是真的能 install / execute / edit 你的电脑 / 浏览器 / IDE」，走 Anthropic / OpenAI / Ollama / 本地模型全兼容；同时配套桌面 GUI、CLI、与 MCP 工具调用。
解决什么：当前 99% 的 Copilot/Cursor/Continue 类工具停留在「建议补全」阶段，要它们「跨文件规划任务、装环境、跑命令、修 bug」就要靠用户手搭。goose 把 Agent 拉到 CLI/桌面一层，给它 MCP 工具、有 plan/edit/test 三段循环、有「会话可审计」机制——OpenAI Operator / Anthropic Computer Use 那一套的高可控、私有部署版。
适用场景：需要让 LLM 真的「能动键盘鼠标」做运维/SRE 私活的企业 IT；想做「Agent 同事」的本地第一方；想做 IDE 工作流之外的 GUI/TUI Agent 跨端操作。
怎么用：安装 goose CLI → `goose configure` 选供应商 → 桌面/CLI 任一入口对话；自带 MCP 客户端，可 `add` MCP server 工具到对话上下文；会话自动落盘可回放。
🔥 近期亮点：Rust 单文件部署；MCP 原生工具调用；Operator-like 本地化

context7 — 让 LLM 拿到「最新」的代码文档
⭐ ~59K
是什么：Upstash 推出的「实时文档上下文服务」——专门解决「LLM 训练数据滞后导致它给你写 2023 年 API」这个老毛病。context7 自动拉各语言库/框架最新版 release-notes、源码注释、API 签名，向 MCP 客户端（Cursor / Claude Code / Cline / Continue）暴露「按需注入 ★最新★ 文档上下文」的能力。
解决什么：开发者每天被 LLM 写的「看起来对、跑起来报错」的过期 API 坑——Next.js 17、React 19、LangChain 1.x、Kubernetes 1.32 的 API 变了它不知道。RAG 自建成本高、维护累；context7 把这件事做成一行 MCP server，注册一次，所有 IDE 通吃。
适用场景：任何「用 LLM Coding 且项目依赖活跃更新的库」的工程师；做企业内 LLM 编码助手的运维（不用再为每个框架单独训练/RAG）；给多 IDE 团队统一一份「文档真相源」。
怎么用：在 Cursor/Claude Code 的 MCP 配置里加 `npx -y context7-mcp`；提示词里加 `use context7`；Agent 调用工具自动拉最新文档示例嵌入上下文；也可在 SaaS 上注册私有库。
🔥 近期亮点：覆盖 2000+ 仓库最新版；Cursor / Claude Code / Cline / Codex 全兼容；私有库可注册

适用场景：家庭实验室 / 中小企业想搭本地「多模态 AI 服务集群」；做 private LLM gateway 给客户演示用兼容 OpenAI 接口；科研机构批量跑视觉/语音/文本多模态评测。
怎么用：curl/brew/docker 装 → `local-ai run` 起服务 → 直接使用 `http://localhost:8080/v1` 的 OpenAI 兼容接口，多模态走 `/v1/images/generations` 等扩展端点；模型从 YAML Gallery 拉。

Unsloth — 让消费级显卡也能微调大模型
⭐ ~68K
是什么：「速度和显存砍半」的 LLM / VLM 微调框架——基于 PyTorch + bitsandbytes，写了高度优化的 kernels：同一个 70B 模型，传统方案要 8×H100，Unsloth 4×A100 搞定；消费级 4090 / 5090 也能微调 7B~32B；同步支持 LoRA / QLoRA / 全参微调，覆盖 Gemma 4 / Qwen 3.6 / DeepSeek / Llama 4 / Phi-4 一众主流开源模型。
解决什么：大模型微调的第一步永远卡在「门槛」：搞环境、配 deepspeed、写 SFT 文件、OOM、调半天跑不起来。Unsloth 把「训练前的所有摩擦」抹平：Notebook 一行启动、显存减半、速度 2-5 倍、内置 GRPO/DPO/SFT/RLHF 配方——让「自己训一个企业专属模型」不再是只有大厂才能做的事。
适用场景：个人研究者用单卡做研究级实验；中小企业在合规场景里训「行业模型 / 客服模型 / 法律模型」；Agent 团队训「Agent 自己的策略模型」做本地化对齐；教学场景快速 demo 微调原理。
怎么用：`pip install unsloth` → 选 `FastLanguageModel.from_pretrained` 加载 4-bit 量化版本；用 `SFTTrainer` / `GRPO` 内置 trainer 配数据；训练完可直接导出 GGUF 给 Ollama、vLLM、LocalAI 部署，也可以 LoRA 合并回原模型走 Hugging Face 推理。
🔥 近期亮点：Gemma 4 / Qwen 3.6 / DeepSeek 全支持；GRPO 内置；单 4090 微调 32B
