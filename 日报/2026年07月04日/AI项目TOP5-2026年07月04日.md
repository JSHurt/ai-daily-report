## 05 Headroom ⭐ 55,832

**本周密集发版：7月3日一天两版**

- v0.29.0 (7/3)：content-router智能分流（无损/有损压缩自动选择）
- v0.30.0 (7/3)：Agent Persona功能（编码/通用场景专属压缩策略）

### 是什么
LLM上下文智能压缩引擎——在工具输出、日志、文件、RAG片段进入LLM之前自动压缩，节省60-85%上下文窗口。新增Persona功能让压缩策略适配不同场景（编码vs写作vs客服），精准度更高。

### 解决什么问题
Agent长线任务导致上下文窗口消耗指数增长——一次多步骤Agent任务可能消耗上百万token。Headroom透明插入压缩层，不改应用代码即可大幅降本，新的Persona功能让压缩更智能。

### 适用场景
- 所有使用LLM API的应用（Agent框架、RAG、代码助手）
- token消耗大、需要降本增效的生产环境
- 希望"无感接入"不改代码的团队

### 怎么用
```bash
pip install headroom==0.30.0
# 在LLM调用链前加：
# content = headroom.compress(raw_content, persona="coding")
```

🔗 [GitHub](https://github.com/headroomlabs-ai/headroom)
