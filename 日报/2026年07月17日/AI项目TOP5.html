screenpipe — 本地录屏记忆，把工作现场交给 Agent
⭐ ~20.2K
是什么：YC S26 开源项目——在本地持续捕获你看到、说到、做到的内容，再变成可搜索记忆、SOP 与自动化素材，供 AI Agent 调用；强调本地、私有、安全，可对接 OpenClaw、Hermes Agent 及 100+ 应用。
解决什么：多数 Agent 只「听你口述」或「读你贴的文件」，看不到真实工作现场：浏览器标签、会议纪要、重复操作路径全丢了。screenpipe 把「你怎么干活」录成可检索上下文，让助手基于真实行为生成流程与自动化，而不是空想 prompt。
适用场景：个人/小团队要给 Agent 补「屏幕记忆」；需要本地优先、数据不出机的知识沉淀；想把日常操作沉淀为 SOP/自动化的效率玩家。
怎么用：官网下载桌面端 onboarding → 开启本地捕获 → 在 Agent/应用侧连接 screenpipe 记忆与自动化接口；本周持续发桌面版（app-v2.5.112，7/15）。详见 screenpipe.com / screenpi.pe。
🔥 近期亮点：app-v2.5.112（7/15）；YC S26；本地私有记忆 + 对接 OpenClaw/Hermes；今日仍在推送
