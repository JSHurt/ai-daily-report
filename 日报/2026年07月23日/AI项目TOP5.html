⭐ 86.9K
是什么：面向大模型的高吞吐、内存高效推理与服务引擎——PagedAttention、连续批处理、OpenAI 兼容 API，覆盖 CUDA/TPU/AMD 等硬件与主流开源模型；稳定线 v0.25.1（7/14 补丁），仓库今日仍在推进量化自动化与 MXFP4 KV cache 等内核优化。

适用场景：私有化/混合云 LLM 推理集群；需要 OpenAI 兼容接口的中台；多卡吞吐优化与量化部署；与 LiteLLM 等网关串联。
怎么用：pip install vllm 或官方 Docker 启动 OpenAI 兼容服务 → 客户端改 base_url；关注 v0.25.x 发布说明与今日量化相关合并。GitHub API 实时：pushed 2026-07-22。
